\# Agricultural Price Analysis Dashboard



\## Overview

This project analyzes 13,000+ records of Maharashtra agricultural market data to uncover price trends, seasonal patterns, and high-performing districts.



\## Tools \& Technologies

\- Excel — Data cleaning

\- Python — Pandas, Scikit-learn, Matplotlib, Seaborn

\- Power BI — Interactive dashboard



\## Key Features

\- Price trend analysis over time

\- Monthly seasonality insights

\- Arrival vs Price relationship

\- Top districts by average price

\- Random Forest price prediction model



\## Machine Learning

\- Model: Random Forest Regressor

\- Features: Arrival quantity, Month, Year, Commodity

\- Evaluation: MAE and R2 Score



\## Key Insights

\- Pomegranate shows highest price trend

\- Higher arrivals generally reduce prices

\- Pune and Amaravati lead in pricing



\## Files

\- `dataset.xlsx` — cleaned dataset

\- `agri\_data\_pipeline.ipynb` — Python analysis and ML model

\- `DASHBOARD.png` — Power BI dashboard preview



\## How to Run

1\. Install dependencies: `pip install -r requirements.txt`

2\. Open `agri\_data\_pipeline.ipynb` in Jupyter

3\. Run all cells
